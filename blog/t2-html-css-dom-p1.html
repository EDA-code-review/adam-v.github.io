-Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML gives the content for a page similar to a word document like headings, CSS gives the style of the page, i.e. changing the colour of the word document or the style of the font and the DOM acts as a platform allowing these changes to form the document.

-What is meant by boxifying design?

You look at a page and think of it's construct as boxes.

-What is the box model?

You build in boxes, developer tools shows you in boxes so you take this approach to building everything in boxes!!
